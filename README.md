# Customer Conversion Analysis Dashboard

## Overview
This project presents an interactive Tableau dashboard focused on customer conversion analysis. It explores customer demographics, call activity, occupation groups, and conversion outcomes to identify patterns that influence campaign performance.

## Key Insights
- Analyzed over 45,000 customer records.
- Measured overall customer conversion performance.
- Examined conversion patterns across age groups and occupations.
- Identified peak customer engagement periods.
- Analyzed call activity trends by day and month.

## Dashboard Features
- Conversion Rate KPI
- Occupation Analysis
- Age Distribution Analysis
- Call Duration Trends
- Monthly Call Activity Analysis
- Conversion Status Comparison
- Interactive Filters

## Tool Used
- Tableau

## Business Value
The dashboard provides insights into customer behavior and campaign effectiveness, helping organizations improve targeting strategies and optimize conversion performance.
