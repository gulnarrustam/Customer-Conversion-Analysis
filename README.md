# Customer Conversion Analysis Dashboard

## Overview
An interactive Tableau dashboard designed to explore customer conversion performance and uncover factors associated with successful marketing outcomes. The analysis focuses on customer demographics, occupation segments, engagement patterns, and campaign activity.

## Highlights
- Analyzed over 45,000 customer records.
- Measured overall customer conversion performance.
- Examined conversion patterns across age groups and occupations.
- Identified peak customer engagement periods.
- Analyzed call activity trends by day and month.

## Tool Used
- Tableau

## Business Value
The dashboard provides insights into customer behavior and campaign effectiveness, helping organizations improve targeting strategies and optimize conversion performance.
